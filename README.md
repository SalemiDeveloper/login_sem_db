# 🔐 Sistema de Login com PHP Puro (Sem Banco de Dados)

Este projeto é um sistema simples de login desenvolvido utilizando **PHP puro**, sem frameworks e sem conexão com banco de dados.

O objetivo principal foi praticar e aprofundar o conhecimento sobre **rotas, controle de requisições HTTP e organização estrutural de aplicações PHP**.

---

## 🎯 Objetivo do Projeto

- Praticar criação de rotas manualmente
- Entender o fluxo de requisição HTTP
- Trabalhar com controle de sessão (`$_SESSION`)
- Simular autenticação sem banco de dados
- Organizar estrutura de aplicação sem framework

---

## 🛠 Tecnologias Utilizadas

- PHP (puro)
- HTML
- CSS
- Servidor local (XAMPP, WAMP ou PHP Built-in Server)

## 🔐 Como Funciona o Login

- As credenciais são simuladas diretamente no código.
- O sistema utiliza sessões (`$_SESSION`) para controlar o estado de autenticação.
- Caso o usuário não esteja autenticado, o acesso às rotas protegidas é bloqueado.
- Não há persistência em banco de dados — os dados são fixos para fins educacionais.

---

## 📚 Conceitos Praticados

- Rotas manuais em PHP
- Separação de responsabilidades
- Controle de sessão
- Redirecionamento
- Tratamento de requisições GET e POST
- Estruturação de aplicação sem framework
